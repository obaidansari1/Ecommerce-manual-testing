# Ecommerce manual Testing Project(Amazon)

#Overview

Manual testing project for an e-commerce web application (Amazon-like system), covering complete user flow from registration to order completion.

## Modules Covered

* User Registration (OTP, validations)
* User Authentication (login, reset, session)
* Search (valid/invalid, filters, suggestions)
* Product Page (details, variants, availability)
* Cart (add/remove/update items, coupons)
* Checkout & Payment (success, failure, validation)
* Orders (order details, cancellation, return & refund)


## Test Artifacts

* Test Cases (Excel)
* Screenshots (execution proof)

## 🚀 Tools Used

* Manual Testing
* Excel
